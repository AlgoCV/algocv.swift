# Agent Workspace

This folder stores reusable agent coordination notes for ImagePro. It is separate
from app source and package source.

Structure:

- `.agents/profiles/codex.md`: default Codex profile for work in this repository.
- `.agents/handoffs/README.md`: format for persistent handoff notes when a task
  needs to be resumed later.

Keep durable coordination notes here. Keep transient command output, build logs,
and large generated files out of the repository.
