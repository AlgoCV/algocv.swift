# Codex Workspace

This folder contains repository-specific guidance for Codex sessions. It is not
part of the app target, Swift packages, or document format.

Start with `ImagePro/AGENTS.md` for the project overview, then use these files
for repeatable Codex workflows:

- `.codex/workflows/change-process.md`: the default change workflow for this
  SwiftUI document app and its local packages.
- `.codex/workflows/validation.md`: validation choices for app, model, ImPro,
  and documentation-only changes.
- `.codex/templates/change-summary.md`: a compact summary template for larger
  changes or handoffs.

Keep this folder focused on agent workflow notes. Do not store secrets,
generated build products, large logs, or task scratch data here.
